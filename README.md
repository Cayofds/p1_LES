# p1_LES
Sobre o projeto:
Este projeto consiste em um site de portfólio desenvolvido para apresentar informações pessoais, habilidades e projetos
O objetivo é criar uma página simples, responsiva e visualmente organizada para demonstrar conhecimentos em desenvolvimento web.

Funcionalidades:
Página inicial com apresentação
Seção “Sobre mim”
Exibição de projetos
Layout responsivo
Navegação simples e intuitiva

Tecnologias utilizadas
HTML5
CSS3
JavaScript
