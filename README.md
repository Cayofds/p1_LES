# p1_LES
Nome e descriçaõ do projeto:
Studido360
Este projeto consiste em um site de portfólio desenvolvido para apresentar informações pessoais, habilidades e projetos práticos de diversos usuários. O principal objetivo é ser um site simples e intuitivo facilitando a utilização para a divulgação de trabalhos pessoais, deis de códigos de programação, mprojetos de marketing até fotografias. 

Tecnologias utilizadas
Front-End:
HTML5
CSS3
JavaScript
Back-end:
PHP (Apenas para conexão com o banco de dados)
Versionamento:
Git
GitHub

Pré-Requisitos:
Visual Studio Code
XAMPP (instalação do banco de dados)
Navegador (Google Chrome, FireFox, Microsoft Edge, etc)

Como instalar:
Instale o XAMPP e o inicie o Apache e MySQL para ter a conexão com o banco de dados

Clone o repositório para a pasta do seu servidor local:
git clone https://github.com/Cayofds/p1_LES

Após abrir a pasta do projeto, clique em Executar e iniciar depuração na página index.html
